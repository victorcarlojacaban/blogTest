### Blog Comment Test

## Setup

#### 1. Clone repository

#### 2. Go to your directory

`cd [your_directory]`

#### 3. Install Composer Libraries

`composer install`

#### 4. Setup Environment

`[create database]`

`[cp .env.example .env file to your directory] and update database configuration`

#### 5. Run migration

`php artisan migrate`

#### 5. Run Seeder

`php artisan db:seed`

#### 7. Run npm

`npm install`

`npm run dev`

#### 9. Launch project on local

`php artisan serve`

#### 10. Access Page

`http://localhost:8000/blogs`
